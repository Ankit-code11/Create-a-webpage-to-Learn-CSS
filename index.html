Chapter 1 · Basics
Colors & Background

CSS mein color dene ke kai tarike hain — naam se, hex code se, rgb se, aur hsl se. Background pe bhi color, image, ya gradient laga sakte ho.

CSS
div {
  color: red;                    /* naam se */
  color: #e50914;              /* hex code */
  color: rgb(229, 9, 20);       /* rgb */
  color: rgba(229, 9, 20, 0.5); /* rgba – transparency */
  background-color: #141414;
  background-image: url('image.jpg');
}

Chapter 2 · Basics
Box Model

CSS mein har element ek box hota hai. Ye box 4 layers se bana hai — Content, Padding, Border, aur Margin. Isko samajhna CSS ka sabse important concept hai.

CSS
div {
  width: 200px;
  padding: 20px;          /* andar ki jagah */
  border: 2px solid red;
  margin: 30px;           /* bahar ki jagah */
  box-sizing: border-box; /* best practice */
}

Chapter 3 · Text
Typography

Text ko style karna — font, size, weight, line-height, alignment. Ye sab CSS typography ke andar aata hai.

CSS
p {
  font-family: 'DM Sans', sans-serif;
  font-size: 16px;
  font-weight: 400;        /* 100–900 */
  line-height: 1.6;
  letter-spacing: 1px;
  text-align: center;
  text-transform: uppercase;
  text-decoration: underline;
}

Chapter 4 · Selectors
CSS Selectors

Selectors se hum batate hain CSS kis element pe lagegi. Element, class, id, pseudo-class — sab hain yahan.

CSS
p               { } /* element selector */
.myClass       { } /* class selector */
#myId          { } /* id selector */
*               { } /* sabko select karo */
p:first-child   { } /* pehla element */
li:nth-child(odd) { } /* odd wale */
a:hover         { } /* mouse aane pe */
div > p         { } /* direct child */

Chapter 5 · Layout
Flexbox

Flexbox se elements ko row ya column mein arrange karna bahut aasaan ho jaata hai. Center karna, gap dena, wrap karna — sab kuch 1 line mein.

CSS
.container {
  display: flex;
  flex-direction: row;        /* row | column */
  justify-content: center;   /* horizontal */
  align-items: center;       /* vertical */
  gap: 16px;
  flex-wrap: wrap;
}

CSS Grid

Grid se 2D layout banate hain — rows aur columns dono control hote hain. Dashboard, gallery, complex layouts ke liye perfect.

CSS

  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}
.item-wide {
  grid-column: span 2;

Chapter 7 · Position
CSS Position

Position property se elements ko exactly kahan rakhna hai wo control karte hain. 5 types hote hain — static, relative, absolute, fixed, sticky.

CSS
position: static;   /* default */
position: relative; /* apni jagah se shift */
position: absolute; /* parent ke andar */
position: fixed;    /* viewport mein fixed */
position: sticky;   /* scroll hone tak relative */
top: 20px; left: 30px;

Chapter 8 · Motion
Transitions

Transition ek property se doosri property mein smoothly jaane ka effect hai. Hover pe color change, size change — sab smooth ho jaata hai.

CSS
button {
  background: red;
  transition: background 0.3s ease;
}
button:hover { background: darkred; }
/* Shorthand */
transition: all 0.4s ease-in-out;

Chapter 9 · Motion
Animations

@keyframes se apni custom animation banao. Spin, bounce, pulse, shake — sab CSS se hi hota hai, JavaScript ki zaroorat nahi!

CSS
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-20px); }
}
.box {
  animation: bounce 1s ease-in-out infinite;
}

Chapter 10 · Styling
Borders & Box Shadow

Border se element ke around line lagao. Box-shadow se 3D depth effect aata hai. Border-radius se corners round hote hain.

CSS
div {
  border: 2px solid red;
  border-radius: 10px;
  border-radius: 50%;     /* circle! */
  box-shadow: 0 4px 20px rgba(0,0,0,0.5);
  box-shadow: 0 0 0 3px red; /* glow */
}

Chapter 11 · Visuals
Gradients

Gradient se ek color se doosre color mein smoothly transition hota hai. Linear (seedha), radial (circle mein) — dono type hote hain.

CSS
/* Linear Gradient */
background: linear-gradient(to right, red, blue);
background: linear-gradient(135deg, #e50914, #f08804);

/* Radial Gradient */
background: radial-gradient(circle, red, black);

/* Multiple stops */
background: linear-gradient(to bottom, #e50914 0%, #141414 100%);

Chapter 12 · Responsive
Media Queries

Media queries se website responsive banti hai — mobile pe alag style, tablet pe alag, desktop pe alag. Aajkal har website mein ye hona chahiye.

CSS
/* Mobile first */
.container { width: 100%; }

@media (min-width: 768px) {
  .container { width: 720px; }
}
@media (min-width: 1024px) {
  .container { width: 960px; }
}
@media (prefers-color-scheme: dark) {
  background: #141414;
}
